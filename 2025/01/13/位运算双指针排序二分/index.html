对于每个测试案例：

    (1 ≤ n ≤ 1000，且 n 为奇数)

    (0 ≤ a[i] ≤ 10^9)

输出描述

对于每个测试案例：

  • 若存在这样的 x，则输出 x。
  • 若不存在，则输出 -1。

输出样例1

解释

  1. 对于第一个案例：

    • 选择 x = 5，得到 b = [7, 0, 7]。

多关键字排序

题目

题目描述

Jakon 家里有一堆未整理的书籍，胡乱地摆在书架上。有一天，身为强迫症的他终于看不下去了，于是他决定整理这堆图书，使其变得整齐一些。

他有 n 本高度、厚度、宽度确定的书籍，他决定按照 高度、厚度、宽度分别作为第一、第二、第三关键字，将这些书籍按降序排列成一排。

输入描述

  • 第一行：一个整数 n，表示书籍的数量。
    (1 ≤ n ≤ 2 × 10⁵)

  • 接下来 n 行，每行包含三个整数 h, s, w，分别表示某本书籍的高度、厚度、宽度。
    (1 ≤ h, s, w ≤ 10⁹)

输出描述

共 n 行，根据题意将所有书籍进行排序后，按照顺序，每行输出一本书籍的高度、厚度、宽度。

输入样例1

输出样例1

排序规则

  • 第一关键字：高度 h，降序排列。
  • 第二关键字：若高度相同，则按厚度 s 降序排列。
  • 第三关键字：若高度和厚度相同，则按宽度 w 降序排列。

答案

bool operator < (const Book &u) const详解

  1. 功能：
    这是一个运算符重载函数，用于定义书籍对象的排序规则。
    当我们调用 sort(p + 1, p + 1 + n) 时，std::sort 会使用此处定义的 < 运算符来比较两个 Book 对象。

  2. 逻辑拆解：

    • 第一段：

      cpp

      复制代码

      if (a == u.a && b == u.b) return c < u.c;

      • 如果当前书 a 和目标书 u 的高度相同，并且厚度也相同，则比较宽度。
      • 返回 c < u.c，表示宽度较小的书排在前面（默认升序）。

    • 第二段：

      cpp

      复制代码

      if (a == u.a) return b < u.b;

      • 如果当前书 a 和目标书 u 的高度相同，但厚度不同，则比较厚度。
      • 返回 b < u.b，表示厚度较小的书排在前面。

    • 第三段：

      cpp

      复制代码

      return a < u.a;

      • 如果高度不同，则比较高度。
      • 返回 a < u.a，表示高度较小的书排在前面。

  3. 排序顺序：

    • 默认按照升序排列（从小到大）。
    • 因此，sort() 排序后需要 reverse()，才能实现最终题目要求的降序排列。

输出结果与原理：

  1. 使用 sort(p + 1, p + 1 + n)，先按高度升序、厚度升序、宽度升序排列。
  2. 使用 reverse(p + 1, p + 1 + n)，将结果反转，实现高度、厚度、宽度的降序排列。
  3. 最终按要求输出书籍信息。

通过 operator < 定义排序规则，结合反转操作，可以高效地实现复杂的多关键字降序排列。
