2.差分數組

$d_i = a_i - a_{i-1}$
$\sum_{j=1}^{i} d_j$

题目描述

给定一个长度为 n 的数组 a，和两个整数 p、q。

先进行 p 次区间加操作：将区间 [l, r] 的数字都加上 x。

再进行 q 次区间查询操作：求出 [l, r] 区间的数字之和。

对于每次区间查询操作，输出结果。

输入描述

第一行三个整数 n、p、q。
$(1 ≤ n ≤ 10^5, 0 ≤ p ≤ 10^5, 0 ≤ q ≤ 10^5)$

第二行 n 个整数表示数组 a。
$(-10^9 ≤ a_i ≤ 10^9)$

接下来 p 行，每行三个整数 l、r、x。
$(1 ≤ l ≤ r ≤ n, -10^9 ≤ x ≤ 10^9)$

接下来 q 行，每行两个整数 l、r。
$(1 ≤ l ≤ r ≤ n)$

输出描述

对于每次区间查询操作，输出结果。
