使用vscode实现轻量的竞赛配置

发表于2025-01-24|更新于2025-01-24|software
|浏览量:

总所周知，vscode的插件以开放著称，那么当你配置完vscode的运行环境之后，有没有什么比较好的插件可以辅助我们进行竞赛呢。

C/C++ Compile Run

C/C++ Compile Run - Visual Studio Marketplace
这个扩展能够实现c++的单文件编译运行，也就是说你不需要每次都新建一个工程文件然后仔细写上“main.cxx”了，相当实用

Competitive Programming Helper (cph)

Competitive Programming Helper (cph) - Visual Studio Marketplace
大名鼎鼎的cph，搭配上浏览器插件“Competitive Companion”Competitive Companion - Chrome 应用商店使用更佳。

可以看见我们的vscode侧栏出现了输入输出内容，我们可以使用它来快速进行自测数据的判断，非常实用快捷

vscode-luogu

vscode-luogu - Visual Studio Marketplace

相当好用的洛谷集成插件，同时支持cph快速跳转
